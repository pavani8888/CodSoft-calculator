#  CODSOFT - Calculator

This project is a **Simple Web Calculator** developed as part of the **CodSoft Web Development Internship (Level 1 – Task 3)**.

The calculator performs basic arithmetic operations such as addition, subtraction, multiplication, and division using **HTML, CSS, and JavaScript**.

---

##  Project Overview

The objective of this project is to build a functional calculator with an interactive user interface.
It demonstrates the use of **JavaScript for handling user inputs and performing calculations** along with HTML and CSS for structure and design.

---

##  Features

* Basic arithmetic operations (+, −, ×, ÷)
* Interactive calculator interface
* Responsive and clean layout
* Button-based input system
* Real-time calculation display

---

## Technologies Used

* **HTML5** – Structure of the calculator
* **CSS3** – Styling and layout
* **JavaScript** – Logic for performing calculations
